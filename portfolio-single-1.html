Project Date JUN 27th, 2023
Visit http://pupo61.cafe24.com
Role BACK-END
FRONT-END
DATABASE
participants 4 PERSON

프로젝트개요

제목 : FULL POWER SALAD(Spring MVC) - 배포중

기간 : 2023-06-01 ~ 2023-06-27

인원 : 총 4명

설명 : 파이널 팀 프로젝트입니다.
        (1)로그인, 로그인 관련 관리자페이지
        (2)정기구독, 정기구독 관련 관리자 페이지
        (3)상품구매, 상품 구매 관련 관리자 페이지
        (4)고객센터, 고객센터 관련 관리자 페이지

        총 4가지로 분류하여 각자 담당하였고
        그 중, 저는 4번을 맡게 되었습니다.

개발내용

- 그린팅을 참고하여 샐러드 정기구독 홈페이지 구현
- 관리자 로그인시 직접 상품 등록,정기구독 상품등록 가능
- 등록과정에서 이미지 업로드, 판매,매입가격 , 상품분류추가 등 기능 포함
- 정기구독 회원,취소회원 확인 가능
- 자주묻는 질문,공지사항 등 일반 고객에게 필요한 정보를 제공하는
  고객센터 게시판 개발

담당기능

담당기능 : 게시판(공지,자주묻는질문,1대1문의)
              관리자페이지(공지,자주묻는질문,1대1문의)                             - 관리자 아이디 비밀번호는 이력서에 기입하였습니다.
              참고 부탁드립니다.-

공지사항게시판
mvc-model2 패턴 게시판 관리자로 로그인 시 공지글 create,delete 가능 공지 작성시 사진첨부-다운로드 가능 공지사항 게시판 디테일 들어가면 이전글,다음글 확인가능 pagination을 적용하여 사용자가 보기 편리하도록 게시판의 게시글 노출갯수를 10,20,30개로 선택하여 볼 수 있습니다.

자주묻는질문
아코디언 형식의 게시판 자주묻는 질문은 고객센터 홈에서 간략하게 드래그로 확인가능 mvc-model2 패턴 게시판

1대1문의
로그인한 회원만 접근 가능하도록 하였습니다. input에 입력된 정보들을 관리자 모드로 로그인하면 관리자페이지에서 볼 수 있도록 개발하였습니다. 문의글 작성시 회원은 첨부파일을 등록할 수 있습니다.

개발환경

Language HTML5, CSS, Java, SQL, JavaScript

Framework SpringBoot

Library JSTL, JQuery, Lombok, MyBatis, Maven, BootStrap

Database MYSQL

WAS Apache Tomcat9

TOOL Spring-Tool-Suite4
